# Claude-line-spacing
A simple chrome extension to increase the line spacing of Claude's responses

# Installation
Since this extension isn't on the chrome web store, you'll need to load it manually as an unpacked extension

1- Clone or download this repository as a zip and extract it to a folder on your computer

2- In your browser, go to chrome://extensions

3- Toggle the Developer mode in the top-right 

4- Click Load unpacked, then select the folder containing the extension files
